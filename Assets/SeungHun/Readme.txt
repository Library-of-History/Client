Chipped Stone : 뗀석기들 (주먹도끼, 슴베찌르개 등)의 fbx 파일 및 material, texture
Fonts : UI에 사용한 한글 폰트
Prefab: 몸돌, 망치돌, 박편, 뗀석기 만드는 과정의 돌, 주먹도끼, NPC 프리팹
Scenes: 개발하던 씬
Scripts : 각종 스크립트들
Scripts 내부 폴더 설명

Book1: 뗀석기 제작 인터렉션과 관련된 코드들
Dialogue: 대화 시스템 관련 코드들
NPC: NPC가 플레이어를 감지하고, 회전하는 것과 관련된 코드 (NPCCharacter는 Dialogue와 연결됨)
VR: VR 입력 관련 / VR 환경에서 플레이어 추적하는 코드

SFX: 뗀석기 제작 중 충돌 시 재생하는 사운드
Stones: 몸돌 과 망치돌, 박편 및 뗀석기 만드는 과정의 돌에 사용된 일반 돌 asset (fbx 파일, material, texture)
